Requirement
===========
1. PHP5
2. MySQL (4.1 or greater)
3. Apache with mod_php
4. PHP gd

Installation
============
1. edit and save 'include/setup.php'
2. run 'http://yourdomain/install' on Webbrowser. It should be noted that 'http://yourdomain/'
   must be the same as ROOT_URI in edited 'include/setup.php'
