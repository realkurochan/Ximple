# Ximple Quick-Guide

## Installation

- Get a host
- Upload Ximple
- Write setup.php

### Get a host

- Apache
- PHP newer than version 5.3
- MySQL or PostgreSQL or SQLite
- It should be great, if it has mod_rewrite (http://httpd.apache.org/docs/1.3/mod/mod_rewrite.html).
